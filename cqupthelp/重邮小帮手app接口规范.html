重邮小帮手App接口规范

Attention:

  1. 所有的请求参数均采用POST方式.
  2. 返回信息中包括有status和info, 若执行错误,status为0且info输出错误信息. 若执行无误,status为200且info为success.

周边

餐厅

1.获取商店列表

url:

请求示例:

{
    "pid"   :1
}

返回示例:

{
    "status":200/0,
    "info":"success,错误信息"
    "data":[
        {
            "shop_id"       :1,
            "shop_name"     :"沁园干锅",
            "shop_image"    :"http://su.bdimg.com/static/xtpl/img/music/fmsprite_35b05295.png",
            "shop_address"  :"堕落街A区"
        },

        ...

        {

            ...

        }
    ]

}

Ps.

1.一个店铺会对应有多张图片,此处返回的shop_image应为第一张照片的url绝对地址

2.pid为页码,每页默认15条数据,每页数据量要求可以在后台管理系统中修改

2.获取商店详情

url:

请求示例:

{
    "shop_id"   :1
}

返回示例:

{   
    "status"            :200/0,
    "info"              :"success/错误信息",
    "data"              :[
        "shop_id"           :1,
        "shop_name"         :"沁园干锅",
        "shop_image"        :{
            "http://alksdj.com/aldkj.jpg",
            "http://asdlj.com/asdlkjalsdj.jpg"
        },
        "shop_address"      :"堕落街A区",
        "shop_tel"          :"13333333333",
        "shop_content"      :"欢迎来到沁园干锅,这里的脆皮鸡很好吃的干活",
        "shop_sale_content" :"脆皮鸡八折了哈!"
    ]

}

Ps.

1.shop_image为url绝对地址数组,里面包含了对应id的店铺全部信息

2.shop_content为店铺简介

3.shop_sale_content为优惠信息

3.评论列表

url:

请求示例:

{
    "shop_id":1,
    "pid":1
}

返回示例:

{
    "status"    :0/200,
    "info"      :"错误信息/success",
    "data"      :[
        {
            "comment_id"        :1,
            "comment_content"   :"这是一条评论",
            "comment_date"      :"2014年8月32日",
            "comment_author_name":"李竞熠"
        },

        ...

        {

            ...

        }   

    ]

}

Ps.

1.pid为页码,每页默认5条数据,每页数据量要求可以在后台管理系统中修改

4.发送评论

url: 请求示例:

{
    "shop_id"               :1,
    "comment_content"       :"你这脆皮鸡吃了要拉肚子啊",
    "comment_author_name"   :"李竞熠"
}

返回示例:

{
    "status"    :0/200,
    "info"      :"错误信息/success",
    "data"      :[ ]
}

5.菜单列表

url: 请求示例:

{
    "shop_id"   :1
}

返回示例:

{
    "status"    :0/200,
    "info"      :"错误信息/success",
    "data"      :[
        {
            "dish_id"       :1,
            "dish_name"     :"脆皮鸡",
            "dish_image"    :"http://www.baidu.com/sb.jpg",
            "dish_price"    :"20.0",
            "dish_praise_count":10
        },

        ...

        {
            ...
        }
    ]
}

Ps. dish_image是一种菜对应所有图片的第一张. dish_price是价格,以浮点数保存. praise_count是点赞数

6.菜品点赞

url: 请求示例:

{
    "dish_id":1,
    "user_number":2012212222
}

返回示例:

{
    "status"    :0/200,
    "info"      :"错误信息/success",
}

Ps.注意判断,每人对一道菜只能点赞一次, user_number是学号
